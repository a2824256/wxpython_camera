# wxpython_camera
wxpython编写的获取摄像头图像程序，支持zed mini

# 使用方法
点击take photo按钮

# Conda环境配置（python 3.7）
安装wxpython
```shell
conda install wxpython
```
安装opencv
```
conda install -c menpo opencv
``` 
安装zed mini驱动<br>
教程:https://github.com/stereolabs/zed-python-api/blob/master/README.md
